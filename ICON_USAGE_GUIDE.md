## Dynamic Icon System

Your portfolio now supports **ALL icons** from the Simple Icons (Si) package and other react-icons packages!

## How to Use Icons

### In Admin Dashboard

When adding or editing a skill, use the icon name in the format:

- **Simple Icons:** `SiApacheairflow`, `SiPython`, `SiReact`, `SiKubernetes`, etc.
- **Font Awesome:** `FaPython`, `FaGithub`, `FaAws`, `FaDocker`, etc.
- **Material Design:** `MdSettings`, `MdCode`, etc.
- **Ionicons:** `IoLogoGithub`, etc.
- **Game Icons:** `GiCode`, etc.
- **Feather Icons:** `FiCode`, etc.

### Finding Icon Names

1. Visit: https://react-icons.github.io/react-icons/
2. Search for the technology/tool you need
3. Click on the icon to see its name
4. Use that exact name in the admin dashboard

### Icon Naming Rules

1. **Simple Icons (Si):** Usually camelCase with technology name
   - Apache Airflow ‚Üí `SiApacheairflow`
   - Apache Kafka ‚Üí `SiApachekafka`
   - Node.js ‚Üí `SiNodedotjs` (dots become "dot")
   - .NET ‚Üí `SiDotnet`

2. **Font Awesome (Fa):** Usually PascalCase
   - Python ‚Üí `FaPython`
   - GitHub ‚Üí `FaGithub`
   - AWS ‚Üí `FaAws`

3. **Case Sensitive:** Icon names are case-sensitive!
   - ‚úÖ `SiApacheairflow` (correct)
   - ‚ùå `Siapacheairflow` (wrong)
   - ‚ùå `siApacheairflow` (wrong)

### Troubleshooting

**Icon not showing?**
1. Check the icon name spelling (case-sensitive!)
2. Verify the icon exists at: https://react-icons.github.io/react-icons/
3. Check browser console for warnings
4. Default icon (FaCode) will show if icon not found

**Common Mistakes:**
- ‚ùå `SiApache Airflow` (spaces not allowed)
- ‚ùå `SiApacheAirflow` (wrong capitalization)
- ‚úÖ `SiApacheairflow` (correct)
